# GAAS: Gamified Adaptive Arithmetic System

An interactive math learning game with campaign mode, arcade challenges, and a built-in calculator.

## Features

- **Campaign Mode**: Progress through 30 levels of Addition, Subtraction, Multiplication, Division, and Visual Counting
- **Arcade Mode**: Score Attack, Endless Survival, Duel, Flash Memory, and Marathon challenges
- **Practice Mode**: Unlimited practice with adjustable difficulty
- **Visual Questions**: Count dots, triangles, and stars
- **Multiple Themes**: Simulation, Paper, Midnight, and Forest themes
- **Timer Modes**: Standard, Relaxed, Speed, Zen, and Custom
- **Leaderboards**: Compete for high scores
- **Achievements**: Unlock achievements as you progress
- **Calculator**: Built-in scientific calculator with history

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the app:
   ```bash
   npm run dev
   ```

## Build for Production

```bash
npm run build
```
